# Chzzk-Recorder-Pro

## Project Overview
A professional recorder for Chzzk streams, built with a modern stack.

## Tech Stack
- **Backend**: Python 3.14 (FastAPI)
- **Frontend**: React 18+ (Vite, TypeScript, TailwindCSS)
- **Deployment**: Windows Batch Scripts

## Getting Started
1. Run `scripts/install.bat` to set up the environment.
2. Run `scripts/start.bat` to launch the application.
